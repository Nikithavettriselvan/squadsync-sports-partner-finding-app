Project Overview:
-The Squashing Sports Partner Finding App is a web application that helps squash players find partners, schedule matches, and manage their match activities. Users can manage their profile, post updates, search for other players, view available matches, and track their own match history.
-This project is developed using HTML, JSON, JavaScript, Python, and Django.

Demo:
You can watch the demo of the project here: [Demo Video](https://drive.google.com/file/d/1Cz2yFx6yKWXwVGpdiTaUcC3xPVcgvEe1/view?usp=sharing)

Dashboard Features:
The dashboard includes multiple sections for user interaction:
- Edit and view profile
- Post images
- Search for other players
- View available matches
- Schedule matches
- View personal match history
- Feedback and About sections

Technologies Used:
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Django
- Data Handling: db.sqlite3

How to Run:

1. Clone the repository:
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

2. Create a virtual environment and activate it:
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

3. Run the development server:
python manage.py runserver

4. Open the browser and visit "http://127.0.0.1:8000"


Notes:
- Make sure to create your own ".env" file to store sensitive information like "SECRET_KEY" before running the project.
- "venv" and database files are excluded from GitHub to keep the repository clean.
